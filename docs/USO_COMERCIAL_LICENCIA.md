# Uso comercial y licencia

Según el README oficial de MuseTalk:

- El código está publicado bajo licencia MIT.
- No hay limitación para uso académico o comercial.
- Los modelos entrenados están disponibles para cualquier propósito, incluso comercialmente.
- Los otros modelos open-source usados por MuseTalk deben respetar sus propias licencias.
- Los datos de prueba del repositorio son para investigación/no comercial.

## Regla práctica para vender videos

Puedes vender videos generados con tu propio material:

- Tu video base.
- Tu audio.
- Tus personajes o material con derechos.
- Modelos descargados oficialmente.

No uses los videos/datos de prueba del repositorio oficial para vender.

## Recomendación

Para negocio, usa:

```text
tu video generado con Wan / ComfyUI / cámara
tu audio TTS o voz propia
MuseTalk para lip-sync
```

Así evitas problemas con derechos de terceros.
